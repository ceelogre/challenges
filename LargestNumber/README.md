# Largest Number
Given an array of numbers, compile the largest number that can be formed and return it.   
Examples: 
1. [1,2,3,4] => 4321
2. [3,2,9] => 932
3. [6,61,68] => 68661
4. [4,42,46,427,465] => 46546442742
2. [5, 52,57, 517, 532,569,581] => 58157569553252517

### How to test
`python3 largest_num.py list`
